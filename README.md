# AudioClassificationProject

## Download dataset

To download the dataset, run the command

```src/download_dataset.sh```

## Process raw data

To process the raw data, run the command
```python3 src/preprocessing.py```

## Generate audio representations

To generating representations of the feautres, run the command
```python3 src/representations.py```

## Extracting features

To extract feeatures, run the command
```python3 src/features.py```
